ENTRENAMIENTOS DE FUERZA

TANTO PARA CALISTENIA Y GYM

FONDOS EN BARRA FIJA

En este ejercicio debemos intentar tocar el pecho con la barra, siempre sacando pecho. Si eres principiante solo aguanta la bajada y poco a poco iras ganando fuerza.

NIVEL PRINCIPIANTE/INTERMEDIO

FLEXIÓN A UNA MANO

Este nivel ya es más avanzado, pero depende mucho de tu peso. Si eres delgado se te resultara más fácil hacer este ejercicio, recordar formar un triángulo con las piernas de apoyo y brazo.

NIVEL AVANZADO

FLEXIÓN ARQUERA

Esta flexión se realiza deslizándote de izquierda a derecha, el pecho tiene que tocar el suelo, si recién empiezas con este tipo de flexión te recomiendo aguantar la bajada.

NIVEL INTERMEDIO/AVANZADO

FLEXIÓN CON RODILLAS

Esta flexión es la primera que debes hacer para progresar poco a poco. Recuerda tocar el pecho con el suelo y subir lo más fuerte posible, es decir explosivamente.

NIVEL PRINCIPIANTE

FLEXIÓN NORMAL

Esta es la flexión común, la que todos conocen. Son muy buenas para progresar en diferentes tipos de trucos o flexiones, es importante tener el abdomen contraído y la espalda recta, sacar pecho y subir explosivamente.

NIVEL PRINCIPIANTE/INTERMEDIO
